CampusWell is an AI-powered wellness assistant for college students.  
It helps manage stress, exam pressure, and emotional well-being using a friendly chat interface.

## Features
- Empathetic AI chat (powered by Groq + LLaMA)
- Mood tracking
- Breathing exercises
- Daily tips
- Exam mode
- Motivation quotes
- Export chat history

## Tech Stack
- Python
- Streamlit
- Groq API (LLaMA model)

pip install -r requirements.txt
